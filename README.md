# Exam System (在线刷题考试系统)

基于 PHP + MariaDB 开发的轻量级、智能化的在线刷题与考试系统。

## 🌟 功能特性

### 👨‍💻 后台管理
- **管理员登录**：前后台 Session 完全分离，安全互不影响。
- **科目与题库管理**：支持科目分类；支持单选题、多选题、判断题、填空题、名词解释、简答题、实操论述题等多种题型。
- **Excel 题库导入**：支持 xls/xlsx/csv 格式一键导入，自动清理冗余 HTML 实体标签和格式。
- **考试与组卷管理**：自定义组卷规则，支持时间窗口和班级分配；支持随机抽题。
- **主观题 AI 辅助评分**：针对填空题、简答题、名词解释等主观题型，提供相似度判定和自动评分参考。
- **学生与成绩管理**：学生账号管理及 Excel 批量导入；查看学生刷题记录、成绩统计和覆盖率分析。
- **参数设置**：可配置页面显示、题目数量、学生防作弊发呆提醒等各项系统参数。
- **系统日志**：记录所有管理员操作，支持多条件筛选。

### 🎓 前台学生端
- **学生认证**：便捷的学号/密码登录。
- **在线考试与刷题**：
  - 倒计时限制与自动交卷。
  - 自动保存答题进度，防断网/防刷新。
  - 页面防复制、防作弊监控机制。
  - 智能无活动（发呆）提醒弹窗。
- **成绩与解析**：考试完成后即刻查看得分、正确率、题目解析及**知识点**（关联考核目标）。
- **错题本与历史记录**：自动收集错题，按科目分类展示，便于针对性复习；记录全量历史成绩和题库覆盖率。

## 🚀 安装部署步骤

### 1. 数据库配置
- 导入根目录下的 `database.sql` （或您的最新备份 `exam_database.sql`）到 MariaDB/MySQL 数据库。
- 修改 `inc/db.inc.php` 中的数据库连接信息：
  ```php
  $pdo = new PDO("mysql:host=localhost;dbname=您的数据库名;charset=utf8mb4", "您的用户名", "您的密码");
  ```

### 2. 安装依赖
确保服务器已安装 Composer，执行：
```bash
composer install
```
*(主要依赖 PhpSpreadsheet 库用于解析 Excel 题库)*

### 3. 目录权限
请确保以下目录对 Web 服务器进程（如 `www`）具有读写权限：
- `uploads/` - 用于临时存储上传的 Excel 导入文件。

### 4. 初始管理员账号
- 默认用户名：`Admin`
- 密码请通过初始化脚本或您的环境配置设置。

## 📑 Excel 题库导入格式规范

系统支持强大的 Excel 导入功能，Excel 文件应严格包含以下列（第一行为表头，从第二行开始读取）：

1. **题目类型**（必须是：单选题、多选题、判断题、填空题、名词解释、简答题、实操论述题）
2. **大题题干**
3. **正确答案**（填空/简答题若为空，系统会自动尝试抓取选项A作为答案）
4. **答案解析**
5. **知识点**
6. **选项A**
7. **选项B**
8. **选项C**
9. **选项D**

## 💡 注意事项与运行环境

1. **PHP 环境要求**：推荐 PHP 7.4 或更高版本。
2. **防作弊机制**：学生端考试页面屏蔽了右键、F12 及常用的复制粘贴快捷键，如需放开需修改 `exam.php` 和 `exam_result.php`。
3. **主观题自动处理**：主观题由于灵活度高，答题时不进行实时前端验证，通过后台相似度函数及老师复核来评分。
4. **页面排序优先级**：试卷题目目前默认将按照 `单选题 -> 多选题 -> 判断题 -> 填空题 -> 名词解释 -> 简答题 -> 实操论述题` 呈现。
5. **Session 隔离**：前后台状态独立，互相切换不冲突。
